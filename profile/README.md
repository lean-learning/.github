# 🏃‍♂️ Lean Learning (린 러닝)

> **"지식을 행동으로, 배움을 가치로."**  
> 린 러닝(Lean Learning)은 배우고 싶은 분야를 가볍게 시작하고, 빠르게 실행하며, 배운 것을 세상과 공유하는 학습 공동체입니다.

우리는 단순한 이론 학습을 넘어, **'만들면서 배우는 것(Learning by Doing)'**의 가치를 믿습니다. 각자의 관심 분야를 학습하고, 이를 통해 MVP(Minimum Viable Product)를 만들어 세상에 공유하며 함께 성장합니다.

---

## 💡 Our Core Values (핵심 가치)

*   **Lean (가볍고 빠르게)**: 장황한 준비보다 우선 한 걸음 내딛는 것을 지향합니다. 최소한의 준비로 시작하고 피드백을 통해 보완합니다.
*   **Build (만들면서 배우기)**: 단순한 입력(Input)에 머무르지 않고, 작은 프로젝트나 MVP 같은 출력(Output)을 통해 배움을 내재화합니다.
*   **Share (공개와 공유)**: 가능하면 Public Repository를 활용하여 우리가 배운 지식과 결과물을 세상과 나누고 소통합니다.

---

## 🛠️ How We Learn (우리의 학습 방식)

1.  **Repository-driven Learning (레포지토리 기반 학습)**
    *   각자 공부하는 주제나 기술 스택을 개인 Repo(또는 Org 내 Repo)에 기록하고 정리합니다.
    *   학습 과정 자체가 하나의 훌륭한 포트폴리오가 됩니다.
2.  **MVP (Minimum Viable Product) 제작**
    *   배운 내용을 바탕으로 실제로 동작하는 가장 가벼운 서비스를 만듭니다.
    *   아이디어를 빠르게 구현해보는 실행력을 기릅니다.
3.  **Open Feedback & Collaboration (열린 피드백과 협업)**
    *   서로의 레포지토리를 방문해 코드를 리뷰하고, 의견을 나누며 자극을 받습니다.
    *   필요에 따라 공동 프로젝트로 발전시키기도 합니다.
